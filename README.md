# aoc-2022

This repository contains my solutions to the Advent of Code 2022.

## Setup

```bash
python -m venv ./.venv

source ./.venv/bin/activate

pip install -r requirements.txt
```

## Format

```bash
black .
```

## Test

```bash
pytest
```

## Run

